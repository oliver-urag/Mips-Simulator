Overview

Eazy is a programming language that is created as part of a requirement of the Advance Programming Language course. It is intended to be  interpreted in a single machine and should be capable of essential programming constructs. A programmer should be able to create basic calculation and input and display programs.

The language is patterned with C and it aims to provide a more readable version. It also integrates a UI input accepting mechanism that is checks the data type of user inputs. It also introduces the “bool”  and “string” data types which are present in the more advanced language than C.

The next sections shall guide the reader the basics on how to define a program using Eazy. The dicussion in this manual uses the Eazy IDE.

 

Program Structure

A program in Eazy contains 4 different sections :

·         Import Section  (optional) – where headers and inclusions are defined

·         Global Variable Definition List (optional)  – where global variables are defined

·         Function Definition List (required) – where functions are defined

The sections’ order should be strictly followed.

Program execution starts with the “main” function. Thus, this function is required to be defined.

Variable Declaration

A variable can be declared by specifing its type then an identifier to name it. In the example below the vaiable ‘counter’ is being declared with a type ‘Integer’. Declarations should be terminated by a semi-colon

( ; ).

int counter

A variable can be initialized immediately after being declared by using the assignment operator ( = ).

int counter = 1; // initialization using a constant

float salary = basePay + (basePay * tax); // initialization by an expression

Global Variables can only be initialized using constants.

 

Arrays

An array is a data structure that holds a sequence of data with same type. An array can be declared by specifying it size and naming it using an identifier. Array declaration should be terminated with semi-colon

 ( ; )

string employee_names[5];

An array can be initialized using the assignment operator followed by an initializer list enclosed in braces

string employee_names[5] = { “John”, “Mary”, “Sam”, “Jason”, “Ana” };

The size of the array need not be specified when using an initializer. The number of items in the initializer list automatically becomes the size of the array.

string employee_names[] = { “John”, “Mary”, “Sam”, “Jason”, “Ana” };

Functions

Defining Functions

To define a function, start using the “function” keyword followed by a user defined identifier. Function identifiers should be unique

function myMethod

This should be followed with an arguments list enclosed in parentheses.  Arguments can be a variable definition or an array definition.

function myMethod(int x, char y, float rates[])  

Argument list can be empty.

function myMethod()

A function can be defined with a return type using the “returns” keyword followed  by the type. This is optional. Function with return types should have a return statement within its code block. Return statement should match the return type of the function

function myMethod() returns int

{

      // codes here

      return 1;

}

 

Overloading Functions

A function with the same name and return type can have 2 signatures with different set of arguments

function myMethod() returns int

 

function myMethod(int x, int y) returns int

 

function myMethod(int x[], int y[]) returns int

Control Statements

Control statements splits the program execution to a different paths.

If Statement

The “if statement” executes a set of codes when the specified condition is satisfied. To define the if statement start with the “if” keyword followed by the condition expression (boolean) then the set of codes to execute enclosed by braces.

if (x > 5) //condition

{

      //code to execute when condition is satisfied

}

 

Else block

An else block can be specified optionally which will be executed when the if condition is not satified.

if (x > 5) //condition

{

      //code to execute when condition is satisfied

}

else

{

      //code to execute when condition is not satisfied

}

 

Chained If-Else

A chain of if-else statements can be specified if there are several conditions and several code blocks to be executed. The check is done from the upper if downward. Else block at the end is optional.

if (x > 5) // first condition to be evaluated

{ }

else if(x < 5) // evaluate second condition if first is “false”

{ }

else if(x == 5) // evaluate third condition if second is “false”

{ }

else // optional

{ }

 

 

Iteration Statements

Iteration statements repeats execution of a code block as long as the condition specified is still true.

While Statement

The while statement checks the condition before executing the code block. To define a while statement, start with the “while” keyword followed by the condition expression (boolean) which is enclosed in parentheses then the code block enclosed in braces.

while (x > 5) // condition to execute code block

{

      // code block

      // if in the initial instance the condition is false

      // this block is never executed

}

 

Do While Statement

The do while statement is a variant of the while statement in which the code block is executed first before checking the condition. This ensures that the code block gets executed at least once. To define a do while statement, start with the “do” keyword followed by the code block enclosed in braces then the “while” keyword followed by the condition enclosed in parentheses. The statement should be terminated with a semi-colon ( ; )

do

{

      // code block (executed at least once)

}while (x > 5); // evaluate after execution

 

 

For Statement

The for statement enables a version where the most common iteration operations (initialization, condition check and incrementation) can be defined in a single statement. To define a for loop start with “for” keyword then the initialization, condition and incrementation statement separated by semicolon and enclosed in parentheses. The order of execution is initialization – condition – code block – incrementation. The initialization  part is only executed on the first iteration.

for(x=1 ; x<5 ; x++)

{ // code block (executed at least once) }

 

 

Break Statement

The break statement is used to prematurely exit the iteration even if the condition statement is still satisfied.

for (x=1 ; x<5 ; x++)

{

      y *= x;

      if (y > 100)

      {

            break; // exit the iteration if y > 100 and x is still < 5

      }

}

 

// continue execution here when break is invoked

 

Continue Statement

The continue statement allows the execution to  go to the next iteration and skip the remaining statements in the code block

for (x=1 ; x<5 ; x++) // continue execution here when continue is invoked

{

      y *= x;

      if (y > 100)

      {

            continue;

      }

      printLine(“hello”); // will not be executed when y > 100

}
